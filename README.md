# iRevolution: A Data-driven Exploration of Apple's iPhone Impact in India using Tableau

## Project Overview

"iRevolution" is a data visualization project that aims to investigate and showcase the profound influence and effects of Apple's iPhone on the Indian smartphone market. Through interactive Tableau dashboards and data stories, this project provides actionable insights into Apple's growth, market strategy, and revolution in India's technology landscape.

## Features

- **Interactive Dashboard:** Explore deep metrics on sales, market share, battery type distribution, and demographic adoption of the iPhone in India.
- **Data Story:** Follow the chronological narrative of Apple's strategic moves, key milestones, and year-over-year revenue impact.
- **Minimalist Web App:** A clean, Apple-inspired Flask web application that beautifully embeds the Tableau visualizations for seamless exploration.

## Tech Stack

- **Frontend:** HTML, Vanilla CSS (Apple-inspired aesthetic)
- **Backend:** Python, Flask
- **Data Visualization:** Tableau Public, Tableau Embed API v3

## How to Run locally

1. Ensure Python 3.x is installed on your system.
2. Clone this repository and navigate to the project directory.
3. Move into the `flask_app` directory:
   ```bash
   cd flask_app
   ```
4. Install the required dependencies:
   ```bash
   pip install -r requirements.txt
   ```
5. Start the Flask server:
   ```bash
   python app.py
   ```
6. Open your browser and navigate to `http://localhost:5000` to view the application.
